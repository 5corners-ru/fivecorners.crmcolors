# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-30

### Added
- Правила раскраски карточек в Канбане по стадии сделки/лида/смарт-процесса (STAGE_EQUALS)
- Правила раскраски карточек и полей по значению поля (FIELD_EQUALS)
- Правила раскраски по приближению даты (DATE_APPROACHING — за N дней)
- Выбор стадии из дропдауна на форме правила (без ручного ввода ID)
- Поддержка воронок сделок: выбор конкретной воронки для правила
- Поддержка смарт-процессов: стадии загружаются динамически по выбранному смарту
- Окраска поля в детальной карточке и в списке
- Окраска карточки в Канбане: работает при загрузке страницы и при перетаскивании
- Административный интерфейс: список правил, форма создания/редактирования
- Двухшаговое удаление с сохранением настроек
